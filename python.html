Variables y Tipos de datosLast updated: 02-05-2023

Las variables son utilizadas para almacenar datos temporales, durante la ejecución de un programa.

Creación de variables

Para crear variables en Python, simplemente se requiere identificarlas con un nombre y asignarles un valor. Además, se puede reasignar su valor cada vez que sea necesario, incluso con un dato de otro tipo.

numero = 2
numero = 7

Algunas consideraciones para nombres de variables:

  • El nombre puede comenzar con guiones bajo o letras, seguido de otras letras, números o guiones bajo.
  • El nombre no debe incluir espacios en blanco.
  • El nombre no debe ser igual a una palabra reservada del lenguaje Python, por ejemplo break, while, class.

Advertencia

En Python, las variables no pueden crearse vacías, es decir, deben tener un valor asignado al momento de su creación.

Consejo

Existe una guía de estilos para escribir código en Python llamada PEP 8

Uso de variables

Una vez creadas las variables, en algún momento será necesario consultar los datos almacenados. Para ello, basta con llamarlas por su nombre en el código.

Considerando lo anterior, también se puede asignar el valor de una variable a otra:

numero_1 = 35
numero_2 = 10
nuevo_numero = numero_1

Tipos de datos

En los ejemplos anteriores, las variables han almacenado valores numéricos, específicamente números enteros. Sin embargo, también es posible emplear otros tipos de datos según sea requerido.

En esencia, existen tres tipos de datos en Python: numéricos, booleanos y cadenas de caracteres.


Numérico

Los datos de tipo numérico se dividen en enteros (int), reales o de coma flotante (float) y complejos (complex):

numero_entero = 5
numero_real = 5.4
numero_complejo = 1+5j

Booleano

Los datos de tipo booleanos (bool) permiten almacenar dos valores: True o False

dato_encontrado = True
usuario_autorizado = False

Cadena de caracteres

Los datos de tipo cadena de caracteres (str), también conocidos como string, permiten almacenar secuencias de caracteres, es decir, texto. Y es preciso incluir la cadena entre comillas dobles o simples.

una_cadena = "¡Hola!"
otra_cadena = 'adiós :('

Entrada y salida de datos

El modo elemental en que los usuarios pueden interactuar con un programa. Pudiendo recibir datos ingresados por el teclado y mostrar resultados o información de interés por la pantalla.

Entrada de datos

Una de las formas más utilizadas para capturar datos ingresados desde el teclado, es la función input()

nombre = input()

Info

La función input() guarda los datos ingresados por el teclado como cadena de caracteres.
