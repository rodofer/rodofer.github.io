Introducción

Python es un lenguaje de programación interpretado de alto nivel, disponible en distintos sistemas y entornos, también es multipropósito, además soporta varios paradigmas de programación. Se caracteriza por su código limpio, simple y elegante.

Bloques de código

En muchos lenguajes de programación, los bloques de código usan delimitadores que son representados por un par de llaves. Además las instrucciones deben terminar en punto y coma.

bloque_a
{
    instruccion_1;
    instruccion_2;

    bloque_b
    {
        instruccion_3;
    }
}

instruccion_4;

En Python, no se necesitan llaves para definir el inicio y fin de un bloque. Tampoco se requiere punto y coma tras cada instrucción. En cambio, cada bloque debe comenzar con dos puntos y las instrucciones deben ir indentadas con cuatro espacios.

bloque_a:
    instruccion_1
    instruccion_2

    bloque_b:
        instruccion_3

instruccion_4

Variables y Tipos de datos

Las variables son utilizadas para almacenar datos temporales, durante la ejecución de un programa.

Creación de variables

Para crear variables en Python, simplemente se requiere identificarlas con un nombre y asignarles un valor. Además, ese valor puede ser cambiado cada vez que sea necesario.

numero = 2
numero = 7

Algunas consideraciones para nombres de variables:

  • El nombre puede comenzar con guiones bajos o letras, seguido de otras letras, números o guiones bajos.
  • El nombre no debe incluir espacios en blanco.
  • El nombre no debe ser igual a una palabra reservada del lenguaje Python, por ejemplo break, while, class.

Advertencia

En Python, las variables no pueden crearse vacías, es decir, deben tener un valor asignado al momento de su creación.

Consejo

Existe una guía de estilos para escribir código en Python llamada PEP 8

Uso de variables

Una vez creadas las variables, en algún momento será necesario consultar los datos almacenados en ellas. Y para esto, basta con llamarlas por su nombre en el código.

Considerando lo anterior, también se puede asignar el valor de una variable a otra:

numero_1 = 35
numero_2 = 10
nuevo_numero = numero_1

Operadores aritméticos

Estos operadores permiten realizar cálculos matemáticos.

Operador Nombre Ejemplo
+ Suma 10 + 1 devuelve 11
- Resta 10 - 4 devuelve 6
* Multiplicación 3 * 3 devuelve 9
/ División 7 / 5 devuelve 1.4
% Módulo 25 % 5 devuelve 0
** Exponente 8 ** 2 devuelve 64
// Cociente 7 // 5 devuelve 1

Y como en matemáticas, también es posible agrupar operaciones usando paréntesis.

numero_1 = 21
numero_2 = 3
numero_3 = 4
promedio = (numero_1 + numero_2 + numero_3) / 3

Tipos de datos

En los ejemplos anteriores, las variables han almacenado valores numéricos, específicamente números enteros. Sin embargo, también es posible emplear otros tipos de datos según sea requerido.

En esencia, existen tres tipos de datos en Python: numéricos, booleanos y cadenas de caracteres.


Numérico

Los datos de tipo numérico se dividen en enteros (int), reales o de coma flotante (float) y complejos (complex):

numero_entero = 5
numero_real = 5.4
numero_complejo = 1+5j

Booleano

Los datos de tipo booleanos (bool) permiten almacenar dos valores: True o False

dato_encontrado = True
usuario_autorizado = False

Cadena de caracteres

Los datos de tipo cadena de caracteres (str), también conocidos como string, permiten almacenar secuencias de caracteres, es decir, texto. Y es preciso incluir la cadena entre comillas dobles o simples.

una_cadena = "¡Hola!"
otra_cadena = 'adiós :('

Conversión de datos

En ocasiones resulta necesario transformar un tipo de dato en otro. A continuación se muestran ejemplos de las funciones más utilizadas para este propósito.

int()

var1 = int(3.14)
# Ahora var1 tiene el valor 3

var2 = int("48")
# Ahora var2 tiene el valor 48

float()

var1 = float(2)
# Ahora var1 tiene el valor 2.0

var2 = float("2")
# Ahora var2 tiene el valor 2.0

str()

var1 = str(7)
# Ahora var1 tiene el valor "7"

var2 = str(4.8)
# Ahora var2 tiene el valor "4.8"

Entrada y salida de datos

Es el modo elemental en que los usuarios pueden interactuar con un programa. Pudiendo recibir datos ingresados por el teclado y mostrar resultados o información de interés por la pantalla.

Entrada de datos

Una de las formas más utilizadas para capturar datos desde el teclado, es por medio de la función input(). Al usarla, el programa se detiene a la espera del ingreso de datos. Luego, estos datos pueden ser guardados en variables y el programa puede retomar su ejecución.

nombre = input()

Info

La función input() guarda los datos como cadena de caracteres, incluso si el usuario ingresa números.

Salida de datos

La función print() facilita la acción de mostrar información por pantalla, recibiendo como parámetro los datos que se desean imprimir.

print("Hola mundo")
# imprime "Hola mundo"

Si la función recibe como parámetro una expresión aritmética, imprime por pantalla su resultado.

print(5 + 5)
# imprime "10"

Tambiém se puede mostrar el contenido de una variable.

mensaje = "¡Bienvenido a Python!"
print(mensaje)
# imprime "¡Bienvenido a Python!"

Asimismo, se pueden combinar datos literales y de variables.

num1 = 15
num2 = 6
resultado = num1 + num2
print(f"La suma de {num1} y {num2} es {resultado}")
# imprime "La suma de 15 y 6 es 21"

Colecciones de datos

Las colecciones de datos son un tipo de dato compuesto que permite almacenar múltiples valores, incluso de diferentes tipos, en una misma variable.

Listas

Las listas son un conjunto ordenado de datos (elementos). En otros lenguajes de programación las listas se conocen como arrays, vectores o arreglos.

Por el momento, se han utilizado variables para almacenar datos individuales, por ejemplo para guardar el nombre de unas frutas.

fruta1 = "manzana"
fruta2 = "uva"
fruta3 = "platano"

Creación de listas

Las listas pueden ser guardadas en variables, y todos sus valores deben ser incluidos dentro de un par de corchetes y separados por coma.

frutas = ["manzana", "uva", "platano"]

Acceso a los elementos

Para obtener los elementos de una lista, simplemente se requiere hacer referencia a la variable que la almacena.

print(frutas)
# imprime ["manzana", "uva", "platano"]

Sin embargo, es factible acceder a un determinado elemento haciendo referencia a su índice. Un número entero que va desde 0 hasta la cantidad de elementos menos 1.

print(frutas[0]) # imprime "manzana"
print(frutas[1]) # imprime "uva"
print(frutas[2]) # imprime "platano"

Modificación de elementos

Los valores asignados a cada elemento pueden ser cambiados, haciendo referencia al índice del elemento para asignarle un nuevo valor.

print(frutas)
# imprime ["manzana", "uva", "platano"]

frutas[2] = "pera"

print(frutas)
# imprime ["manzana", "uva", "pera"]

Agregación de elementos

Por medio del método .append() se agrega un nuevo elemento al final de una lista.

print(frutas)
# imprime ["manzana", "uva", "pera"]

frutas.append("fresa")

print(frutas)
# imprime ["manzana", "uva", "pera", "fresa"]

A través del método .insert() se agrega un nuevo elemento en una posición dada (índice).

print(frutas)
# imprime ["manzana", "uva", "pera", "fresa"]

frutas.insert(2, "kiwi")

print(frutas)
# imprime ["manzana", "uva", "kiwi", "pera", "fresa"]

Eliminación de elementos

El método .pop() elimina el último elemento de una lista.

print(frutas)
# imprime ["manzana", "uva", "kiwi", "pera", "fresa"]

frutas.pop()

print(frutas)
# imprime ["manzana", "uva", "kiwi", "pera"]

Del mismo modo el método .pop() puede eliminar un elemento en particular si se precisa su índice.

print(frutas)
# imprime ["manzana", "uva", "kiwi", "pera"]

frutas.pop(2)

print(frutas)
# imprime ["manzana", "uva", "pera"]

Info

El método .pop() además de eliminar un elemento, devuelve el elemento borrado, en caso que sea necesario conocerlo.

Control de flujo

El uso de operadores lógicos y relacionales, junto a estructuras de control condicionales e iterativas, permiten que un programa piense y decida por sí mismo.

Operadores relacionales

Estos operadores también se conocen como operadoores comparativos, ya que facilitan la evaluación de la relación entre dos expresiones, devolviendo un valor booleano como resultado, según corresponda.

Operador Nombre Ejemplo
== Igual a 10 == 10 devuelve True, 10 == 12 devuelve False
!= Distinto que 1 != 2 devuelve True, 2 != 2 devuelve False
> Mayor que 4 > 3 devuelve True, 4 > 4 devuelve False
< Menor que 9 < 10 devuelve True, 10 < 9 devuelve False
>= Mayor o igual que 14 >= 14 devuelve True, 14 >= 15 devuelve False
<= Menor o igual que 20 <= 21 devuelve True, 21 <= 20 devuelve False

Operadores lógicos

Estos operadores sirven para combinar expresiones lógicas para crear condiciones más complejas.

Operador Descripción Ejemplo
and Devuelve un valor booleano verdadero (True) si todas las condiciones se cumplen. En caso contrario, devuelve un valor booleano falso (False). 14 < 15 and 1 == 1 devuelve True
or Devuelve un valor booleano verdadero (True) si por lo menos una de las condiciones se cumple. En caso contrario, devuelve un valor booleano falso (False). 5 == 7 or 9 != 10 devuelve True
not Devuelve un valor booleano opuesto al valor devuelto por una condición. not 25 != 25 devuelve True

Sentencia IF

Se trata de una estructura que evalua una condición. Si ésta se cumple, se ejecutan las instrucciones dentro del bloque.

instruccion_1

if CONDICIÓN:
    instruccion_2

En el ejemplo anterior, existen 2 instrucciones. La primera se ejecutará sin problema. Sin embargo, instruccion_2 se encuentra sujeta al cumplimiento de una condición para ser ejecutada.

A continuación, un ejemplo concreto utilizando el operador relacional >=.

edad = 22

if edad >= 18:
    print("¡Hola, bienvenido!")

Adicionalmente, existe un complemento a la sentencia if llamado else. Este bloque agrupa instrucciones que se ejecutarán cuando la condición del if no se cumple.

edad = 17

if edad >= 18:
    print("¡Hola, bienvenido!")
else:
    print("Debes ser mayor de edad")

La construcción anterior (if-else) sirve cuando el programa tiene dos posibles caminos que recorrer, si no es uno entonces es el otro. No obstante, es probable que se requiera diseñar un algoritmo con más de dos flujos.

Python pone a disposición el bloque elif, que también evalúa una condición.

if fruta == "manzana":
    print("Has elejido Manzana")
elif fruta == "naranja":
    print("Has elejido Naranja")
elif fruta == "platano":
    print("Has elejido Platano")
else:
    print("No tenemos la fruta indicada")

Es posible crear condiciones más complejas con la ayuda de operadores lógicos.

if fruta == "manzana" and total_manzanas > 0:
    print("Has elejido Manzana")

Info

if, elif y else son bloques de código, por lo tanto su contenido debe ir indentado.

Ciclo WHILE

Es una estructura de control iterativa que permite la ejecución de una serie de instrucciones en forma repetitiva, mientras (while en inglés) se cumpla una condición.

while CONDICIÓN:
    instrucciones

En el siguiente ejemplo se utiliza la condición contador < 3 para determinar si las instrucciones dentro del bloque se pueden ejecutar, similar al comportamiento de la sentencia IF. Pero en lugar de ejecutar las instrucciones una sola vez, estas podrían repetirse si la condición se sigue cumpliendo, o terminar con el ciclo cuando la condición deje de cumplirse.

contador = 0

while contador < 3:
    contador += 1
    print(contador)

A continuación se representa la ejecución del código anterior, comenzando con la instrucción contador = 0 para asignar un valor inicial a la variable. Luego el programa llega al inicio del bloque WHILE para evaluar la condición contador < 3. Y como contador es igual a cero (inicialmente), la condición se cumple porque cero es menor que tres. En seguida se ejecuntan las instrucciones dentro del bloque, primero contador += 1 que suma uno al valor actual del contador, y print(contador) que imprime el valor modificado de esa variable, dando lugar a la primera iteración.

En efecto, al tratarse de un bloque repetitivo, después de ejecutar las instrucciones en la primera iteración, se evalua nuevamente la condicón en el inicio del bloque. La variable contador ahora tiene el valor uno, y la condición continua cumpliendose porque uno es menor que tres, permitiendo así la segunda iteración.

Como es de esperar en un ciclo iterativo, de nuevo la condición es evaluada. La variable contador ahora vale dos, por lo tanto la condición vuelve a satisfacerse ya que dos es menor que tres, ocasionando la tercera iteración.

Luego de la tercera iteración, el valor de la variable contador es tres, y la condición deja cumplirse porque tres no es menor que tres, sino que es igual a tres. Entonces el ciclo termina.

Variable contador Condición print()
1° intento 0 0 < 3 = True 1
2° intento 1 1 < 3 = True 2
3° intento 2 2 < 3 = True 3
4° intento 3 3 < 3 = False

Advertencia

Si el bloque WHILE está sujeto al cumplimiento de una condición que no cambia, podría generarse un ciclo infinito.


Sentencia BREAK

Un ciclo puede terminar antes de lo esperado si se ejecuta la instrucción break, sin importar que la condición del bloque se cumpla o no.

contador = 0

while contador < 3:
    contador += 1
    if contador == 2:
        break
    print(contador)

Setencia CONTINUE

También se puede interrumpir una iteración en particular si se incluye la instrucción continue, provocando que termine la iteración en curso y comience la siguiente.

contador = 0

while contador < 3:
    contador += 1
    if contador == 2:
        continue
    print(contador)

Ciclo FOR

Es una estructura de control iterativa para ejecutar instrucciones en forma repetitiva, y se diferencia del ciclo WHILE porque el número de repeticiones es conocido. Además, los ciclos no están sujetos a una condición.


Operador IN

for ELEMENTO in ITERABLE:
    instrucciones


lista_numeros = [8, 2, 10, 5]

for numero in lista_numeros:
    print(numero)
Variable numero
1° iteración 8
2° iteración 2
3° iteración 10
4° iteración 5

Función RANGE()

for i in range(4):
    print(i)

Variable i
1° iteración 0
2° iteración 1
3° iteración 2
4° iteración 3 

for i in range(5, 9):
    print(i)

Variable i
1° iteración 5
2° iteración 6
3° iteración 7
4° iteración 8 
for i in range(10, 17, 2):
    print(i)

Variable i
1° iteración 10
2° iteración 12
3° iteración 14
4° iteración 16
